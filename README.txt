ИНСТРУКЦИЯ ПО ЗАПУСКУ WEATHER SYSTEM

1. ТРЕБОВАНИЯ
- Установленная среда Java JDK 17 или выше.
- Установленный Apache Maven для сборки проекта.

2. СБОРКА И ЗАПУСК
- Откройте терминал в папке проекта.
- Для компиляции и запуска используйте команду:
  mvn clean javafx:run

3. ИСПОЛЬЗОВАНИЕ
- При запуске откроется окно авторизации. Введите логин и пароль.
- Если у вас нет аккаунта, нажмите кнопку "Регистрация".
- После входа в систему введите название города в строку поиска и нажмите "Найти".
- Чтобы получить отчет о погоде, нажмите кнопку "Экспорт отчета". Файл будет создан в папке проекта.

4. ЖУРНАЛ ДЕЙСТВИЙ
- Все действия пользователя записываются в файл app.log в папке проекта для целей аудита.
